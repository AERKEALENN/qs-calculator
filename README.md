# QS Calculator

A dark glassmorphism desktop calculator built with Tauri 2.0 + React + TypeScript.

## Features

- Dark glassmorphism UI with dynamic aurora canvas background
- Mouse-following specular highlight on glass surfaces
- Portrait and landscape responsive layouts
- Calculation history (landscape mode)
- Keyboard support
- Custom titlebar (minimize, maximize, close)

## Prerequisites

- [Node.js](https://nodejs.org/) >= 18
- [Rust](https://www.rust-lang.org/) >= 1.70
- [Tauri CLI](https://v2.tauri.app/start/prerequisites/)

## Getting Started

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```
